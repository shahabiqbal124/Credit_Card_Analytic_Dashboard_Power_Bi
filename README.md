An interactive Power BI analytics project that explores credit card transaction and customer data to uncover revenue trends, spending patterns, and customer segmentation insights. The solution includes transaction-level and customer-level dashboards, leveraging Power Query and DAX for data transformation, KPI calculations, and time-based analysis.

**Key highlights:**

Revenue and interest analysis across card categories and transaction modes

Quarterly transaction and revenue trend analysis

Customer demographic segmentation by income, occupation, age, gender, and region

Top-performing states and customer groups by revenue contribution

**Tools:** 
Power BI Desktop, 
DAX, 
CSV datasets
